post_sawdust.check_split_read(list_of_fragments)¶

Takes a list of samdicts returns info dict with combinatorial status Insures that the fragment of split reads can be combined into one SAM line. The following checks are carried out:

  • Same chromosome
  • Same strand orientatien +/-
  • userdefined maximum distance allowed between fragments
  • No overlap between fragments
  • No switchorder within fragments
